# Documentation Map

Use this index to find the right reference quickly:

- `ai-assistants/` – Guidelines for LLM copilots (`AGENTS.md`, `CLAUDE.md`, etc.).
- `plans/` – Active plans plus `plans/archive/` for historical feature roadmaps.
- `reports/` – Post-mortems, investigations, and analytics summaries.
- `runbooks/` – Operational guides split into `deployment/`, `operations/`, and `qa/`.
- `guides/`, `setup/`, `exploration/` – Deep dives on tooling, onboarding, and research spikes.
- `legacy/` – Archived documents that remain for historical context.

Not sure where to start? Follow the runbooks for day-to-day procedures and the plans folder for in-flight initiatives.
